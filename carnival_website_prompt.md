# Prompt: Carnival Celebration Website

Create a vibrant, festive single-page website announcing the arrival of Carnival celebrations. The website should capture the energy, joy, and cultural richness of Carnival.

## Requirements

### Design & Colors
- Use the Brazilian flag color palette:
  - **Green** (#009c3b) - primary accents, headers
  - **Yellow** (#ffdf00) - highlights, call-to-action buttons
  - **Blue** (#002776) - backgrounds, secondary elements
  - **White** (#ffffff) - text on dark backgrounds, clean spaces
- Festive, energetic design with confetti or feather motifs
- Responsive layout for mobile and desktop

### Images
- Use Unsplash images (https://unsplash.com) for:
  - Hero section: Carnival parade with dancers in colorful costumes
  - Gallery section: Street parties, samba dancers, carnival masks
  - Background elements: Confetti, feathers, musical instruments
- Suggested Unsplash search terms: "carnival brazil", "samba dancer", "rio carnival", "carnival mask", "street party"

### Multilingual Support
- Implement language selection (toggle/dropdown) for:
  - **Portuguese (PT)** - default language
  - **Spanish (ES)**
- All text content must be translated between both languages
- Use JavaScript to switch languages without page reload

### Content Sections
1. **Hero Section**
   - Large carnival image background
   - Headline: "O Carnaval Chegou!" / "¡El Carnaval Llegó!"
   - Subheadline with dates and location
   - CTA button: "Saiba Mais" / "Saber Más"

2. **About Section**
   - Brief history and significance of Carnival
   - Cultural importance in Brazil and Latin America

3. **Events Schedule**
   - List of main events with dates, times, locations
   - Street parades, blocos, samba school performances

4. **Gallery**
   - Grid of carnival images from Unsplash
   - Lightbox effect on click

5. **Footer**
   - Social media links
   - Contact information
   - Language selector

### Technical Requirements
- Single HTML file with embedded CSS and JavaScript
- Clean, semantic HTML5 structure
- CSS animations for festive effects (floating confetti, subtle movements)
- Smooth scroll navigation
- Mobile-first responsive design
- No external dependencies except Unsplash images
